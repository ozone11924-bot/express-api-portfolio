# Express API + Playwright Test Suite

A fully tested Express.js API demonstrating junior‑level backend engineering and QA automation skills. The project includes a complete in‑memory data model, authentication, role‑based access control, pagination, filtering, sorting, and a comprehensive Playwright test suite with **40 passing tests**.

---

## Overview

This project showcases a production‑style API built with Node.js + Express, paired with a Playwright test suite that validates both API and UI behavior. It is intentionally designed to demonstrate practical skills expected from a junior QA automation or backend engineer:

- Building and documenting REST APIs  
- Implementing authentication and RBAC  
- Writing automated tests  
- Debugging and stabilizing environments  
- Ensuring consistent, reproducible test results  

The API uses in‑memory data (no database required) to keep the project simple, portable, and easy to run.

---

## Features

- JWT Authentication (admin + user roles)
- Role-Based Access Control
- Users CRUD
- Transactions CRUD
- Pagination, Filtering, Sorting
- 30 Seeded Transactions
- Health & Status Endpoints
- Homepage with Title (UI Test Target)
- 40 Playwright Tests (API + UI)
